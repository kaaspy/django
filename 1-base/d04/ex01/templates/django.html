Django

What is Django ?

Django is a high-level Python web framework that enables rapid development of secure and maintainable websites. Created in 2003-2005, it follows the "batteries-included" philosophy, providing built-in tools for common web development tasks like authentication, database management, templating, and URL routing.

Caracteristics

  • Free and open-source
  • MVT architecture (Model-View-Template)
  • Emphasizes security (protection against SQL injection, XSS, CSRF)
  • Promotes DRY principle
  • Scalable
  • Includes admin interface automatically generated

Django's motto: "The web framework for perfectionists with deadlines."

History

The origins of Django (2003-2005)

Django was created in the autumn of 2003 by web programmers at the Lawrence Journal-World newspaper in Kansas, USA. The two original creators were Adrian Holovaty and Simon Willison They began using Python to build web applications for the newspaper's websites. Jacob Kaplan-Moss was hired early in Django's development, joining the team shortly before Willison's internship ended.

Why was Django created ?

The team was building multiple newspaper websites and kept writing the same code over and over again. They identified common patterns and shared code snippets across projects, eventually refining them into a reusable framework. This framework helped them create complex, database-driven websites more efficiently.

The name Django

The framework was named after the legendary Romani jazz guitarist Django Reinhardt. Adrian Holovaty was a jazz guitar player and a big fan of Reinhardt's music, which inspired the name. This choice reflects the spirit of freedom and creativity the developers wanted to infuse into their project.

Public release (2005)

In July 2005, the framework was released publicly under the BSD license (an open-source license). This allowed the open-source community to contribute significantly to its development.

Django Software Foundation (2008)

In June 2008, it was announced that a newly formed Django Software Foundation (DSF) would maintain Django going forward. The DSF is a US-based 501 non-profit organization independent of the original creators, established to support the framework's development, promote its use, and sustain its growing open-source community.
