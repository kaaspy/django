Django template language (DTL)

How the template engine works ?

  • Loading : Django finds the template file
  • Parsing : Scanning for tags
  • Rendering : Takes the context and replaces the tags with passed context values

Core syntax

  • Variables {% templatetag openvariable %} variable {% templatetag closevariable %}

    Outputs the value of variable. Can use dot notation for attributes, dict keys or list index.

  • Filters {% templatetag openvariable %} variable|filter {% templatetag closevariable %}

    Modify or format the output of a variable. Multiple filters can be chained with .

  • Template tags {% templatetag openblock %} tag {% templatetag closeblock %} {% templatetag openblock %} endtag {% templatetag closeblock %}

    Performs logic, loops, include of other templates, etc... The most common tags are :

    • if : conditional
    • for : loop
    • extends : inheritance on templates
    • include : reusage of html
    • load static : usage of static files
    • csrf_token : CSRF protection
    • url : url generation

Variable syntax

{% templatetag openvariable %} variable {% templatetag closevariable %}

Allows you to display the value of a variable. You need to pass it via the context dict when rendering it from the view.

Block syntax

{% templatetag openblock %} block blockname {% templatetag closeblock %} {% templatetag openblock %} endblock {% templatetag closeblock %}
{% templatetag openblock %} extends filename {% templatetag closeblock %}

Allows to define blocks and define their content via extend tag. This page is generated from a base.html file containing the content block. The text you are reading is contained inside this content block. Django generates the complete HTML response by combining those blocks.

Loop syntax

{% templatetag openblock %} for item in items {% templatetag closeblock %} {% templatetag openblock %} endfor {% templatetag closeblock %}

Allows you to loop over a collection of items and use a scoped variable inside it.

Conditional syntax

{% templatetag openblock %} if condition {% templatetag closeblock %}
{% templatetag openblock %} elif subcondition {% templatetag closeblock %}
{% templatetag openblock %} else {% templatetag closeblock %}
{% templatetag openblock %} endif {% templatetag closeblock %}

Allows you to control the display with conditions.
